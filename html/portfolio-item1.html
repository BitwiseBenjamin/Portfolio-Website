<p class="subtitle" Portfolio Website </p>

To begin this project, I dove into several FreeCodeCamp website building videos to gain a solid understanding of website development. From there it was stackOverflow, ChatGPT, and YouTube that helped me create what you see before you.

For the landing page of the website, I kept the color and design neat and professional. The use of the texting animation is the highlight of this page, and sets the tone and style of the rest of the site.

Here you can see that the animation was acheived by encasing the text in a paragraph element with the class .anim-typewriter. You can also note that to achieve the different color words and html layout the use of & lt; and spans is implimented throughout the project.

In this picture you can see that @keyframes typewriter animation is used within the .anim-typewriter class to give off the typing effect. The @keyframes blinkTextCursor is used in the animation to acheive the blinking cursor.

There was a lot of difficulty in this project when trying to figure out how to adapt this animation to each situation. It was ultimatly decided to apply some different animation settings to each class/area of text.

For the Education, Internship, and Skills section, most of the time was spent finding a way to display the information aestheticly, and adding a download link for my resume for convenience.

The general structure for each section is shown above. ul elements are used to display the information, but two div's within a div are used to dictate the structure of the information.

For the portfolio section of this project, a grid with images representing each project was the best approach to keep the design simple, yet elegant.

Above you can note the use of hover with the transform and transition functionalities, giving off the zoom in and color change effect when hovering over the image.

For the footer the emphasis was put on making it look clean, and adding in stylish logos for the socials.

Above you can see the html structure of the footer, and its use of anchor elements with i elements to provide a link and display the company logo thanks to fontAwesome.

Lastly is the Hamburger and nav elemtents that allow you to navigate through the website.

Above there is the .nav-open class which is called on using javascript. This opens the nav widow, which become the whole page. There is also the code above that shows the transformation put on the hamburger when it is clicked, so it turns into an X shape.

The Javascript above is used to activate the .nav-open class when the hamburger is clicked on, using the event listener.
