<h2 class="subtitle" Hero Creation and Team Building GUI </h2>

This is one of the first major coding projects I created. It taught me many of the fundimentals of computer programming and inspired me to continue my CompSci journey, despite the difficulties.

The picture above shows the first page of the project. The GUI template and format was made using JavaFX. There are two distinct sections in this Add Hero page, the first is the left side that allows for input of a hero and their statistics. The right side will display the different heros that you have inputted into te program. The top combo box for hero type is a dropdown box that allows you to select your hero type, mage, fighter etc...

One of the fundimentals this project taught me is the class. The class is very usefull in computer science, here you can see the initialisation of the class data. Then you can see the public Hero(), which is a constructor for the class and creates the object. Below this you can see me implement an overloaded constructor that creates a hero object in a different way, based off the input given.

Something cool that I picked up durring this project is how to use and create eventhandlers. These are used to determine what you want to transpire if some event were to occur. For the random button handler above, its purpose is generating a random integer between 50 and 100. This will become the attack level of the hero.

Here you can see the Create Army page of the GUI. Its functionality is to load all the heros you created from the Hero Page and give you the option to "add" them to you team, thus combining their statistics and creating your army!

Another important concept exception handling, is shown above. Here is my for loop implimentation of an exeption handler that is thrown when you create two heros with the same name.

Lastly is the code I wrote to create the hero type combo box. This implimentation used the combobox methods to create the title, set its action, and its elements.
