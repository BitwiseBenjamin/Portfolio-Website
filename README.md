## Personal Portfolio Website

## Project Overview
The Personal Portfolio Website is a showcase of my skills, education, experience, and projects. It provides an in-depth overview of my background as a computer science student and my dedication to technology and innovation. The website comprises various sections, including Home, My Experience, About Me, and My Portfolio.

## Description
The website is developed using HTML, CSS, and JavaScript. It leverages external libraries and resources such as Normalize.css, Font Awesome, and Google Fonts for enhanced styling and iconography. The main entry point for the website is `index.html`.

## Sections
The website consists of the following sections:

1. **Home**: This section features an engaging typewriter effect and showcases a picture of me.
2. **My Experience**: This section highlights my educational background, internships, and skills.
3. **About Me**: This section provides insights into my personality, passions, and interests.
4. **My Portfolio**: This section showcases a collection of my projects, which can be accessed by clicking on the project images.

## Usage
To view the website locally, simply open the `index.html` file in a web browser. Navigation across different sections can be done using the top navigation bar.
