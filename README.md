## Personal Portfolio Website

## Project Overview
The Personal Portfolio Website is a showcase of my skills, education, experience, and projects. It provides an in-depth overview of my background as a computer science student and my dedication to technology and innovation. The website comprises various sections, including Home, My Experience, About Me, and My Portfolio.

<img width="400" alt="Screenshot 2023-06-12 at 11 15 08 PM" src="https://github.com/BitwiseBenjamin/Portfolio-Website/assets/114360780/0ad5f7e3-007d-4423-bd33-0b01b7430f86">

## Description & Technologies
The website is developed using HTML, CSS, and JavaScript. It leverages external libraries and resources such as Normalize.css, Font Awesome, and Google Fonts for enhanced styling and iconography. The main entry point for the website is http://161.35.104.92 which is hosted and maintaind on DigitaOcean.
